### 2. Section 01: Hero Banner (main_banner_01)
**크기**: 1920 x 900px

#### 구조
```
main_banner_01
├── 배경 이미지 (디어스치과-0008, 필터 적용)
├── 그라데이션 오버레이 (좌→우 페이드)
├── 의료진 이미지 3개 (절대 위치)
│   ├── 장재원 원장 (중앙)
│   ├── 한정화 원장 (좌)
│   └── 김나현 원장 (우)
├── 텍스트 영역 (좌하단)
│   ├── "디어스 치과" 타이틀
│   ├── 슬로건 텍스트
│   └── CTA 버튼
└── 의료진 정보 카드 3개 (하단)
```

#### 구현 포인트
- 배경: `exposure: 0.33`, `saturation: -0.23` 필터
- 그라데이션: `linear-gradient(to left, transparent 30%, #F5F4F2 100%)`
- 의료진 이미지: 절대 위치, 하단 정렬
- 정보 카드: 흰색 배경, padding 20px 16px, gap 4px

#### 필요 이미지
- `img/main_banner.jpg` (배경)
- `img/main_doctor_img.png` (의료진 합성 이미지)
